About

unifi-cam-proxy adds UniFi Protect compatibility to non-UniFi cameras. It goes beyond the basic third-party camera functionality of UniFi Protect for ONVIF Cameras added by Ubiquiti, key features:

  1. Simulates a Ubiquiti camera, in order to work with Protect

  2. Motion detection allows to record only when motion is detected

  3. Smart detections record when people or cars are detected, for example

  4. Custom support for certain cameras, like Hikvision

  5. Supports any RTSP camera, even without ONVIF support

  6. Allows to create multiple instances, in case you want to add multiple NVR/DVR cameras to protect, as with current ONVIF support only a single camera per ONVIF device is supported
